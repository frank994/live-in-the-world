# 长江大桥

一百年前

一条江

从北至南

由西向东

高山阻拦他

他把他们劈开

成为峡谷

河流阻拦他

他将他们并入



而你

伫立在江上面

一百年



人们好奇

这是个啥

怯



当第一个人从你身上踏过

当第一辆车从你身上驶过

列车和轮船

他们响着汽笛

向你致敬

人们欢呼雀跃

庆贺你

感恩你



一百年了

如今人来人往

汽车、列车、轮船

川流不息

人们忙于赶路

赶着工作

赶着吃饭

赶着回家

不愿抬头看一看你

习以为常

你不愠不怒

习以为常



而我今天要称赞你

你是中国的奇迹

跨长江大桥

公铁一体

中国的骄傲

我愿用赤裸的身体与你联系

愿去了解你

愿去聆听你

愿你

长命百岁

多个百岁




2019-06-22 于武汉
