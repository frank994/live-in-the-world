# 橙子

​			大家好，我叫橙子

- 跟大家说说你自己吧

  嘿，我有什么好说的，就一单板滑手，滑了小十来年了吧。

- 为什么喜欢滑雪

  你不知道啊，接触了滑雪以后，你才发现，卧槽，有这么爽的一个运动。从山上滑下来，穿过树林、蘑菇，你要全神贯注盯着眼前，同时又要全身绷紧地控制你的板子，就那个时候你需要花光你全身的力气和精力去滑。累是真的累，但滑下来以后，你就觉得，卧槽，老子这么牛逼，哈哈哈哈哈			

- 滑雪的时候什么感觉？

  爽！ 就是爽！ 还是爽！

- 听说你最近在学双板？

  对，哈哈哈，双板自由式才是滑雪的归宿！

- 不是骨科医院吗？

  不，是双板自由式 + 大山
