# 巴郎山垭口滑雪计划

大山是每一个滑雪者的最终目标

趁着川西春雪还在，冲！



计划时间：04.23 ～ 04.24

当时天气：

​			小雪，应该有云海，能见度一般，雪质好。

![image-20220417230323487](./image-20220417230323487.png)

积雪深度：100 + 18 ～= 120cm左右

雪层判断：底部是糖霜雪，有雪崩风险



## 行程计划

周五晚上从成都开车到卧龙，当天在卧龙休息。

23号一大早从卧龙开车去垭口顶部（1个半小时左右），爽滑到垭口四姑娘山方向垭口底，随后车接回垭口顶。

晚上回卧龙镇子睡一晚。

24号一大早重复23号爽滑，晚上看情况开回成都或者继续住卧龙。
